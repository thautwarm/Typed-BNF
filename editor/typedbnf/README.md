# TypedBNF

Supported Features:
- Syntax Highlighting
- Go to references (for grammar symbols)
- Go to definitions (for grammar symbols)
- Symbol Outline

For more information, please refer to the [GitHub repository](https://github.com/thautwarm/Typed-BNF).
